## 4장 반복문 요약

반복문은 같은 작업을 여러 번 실행할 때 사용한다. 반복적이고 단순한 작업은 컴퓨터가 매우 잘 처리하므로 자동화에 적합하다.

반복문에는 크게 두 가지가 있다.

for문은 반복 횟수를 알고 있을 때 사용한다.
```
for i in range(5):
    print("방문을 환영합니다!")
```
range() 함수는 일정한 범위의 숫자를 만들어준다.
```
range(5)        # 0부터 4까지
range(1, 6)     # 1부터 5까지
range(1, 10, 2) # 1, 3, 5, 7, 9
```
while문은 조건이 참인 동안 반복한다.
```
while count < 5:
    print(count)
    count += 1
```
반복문에서는누적합, 카운트, 숫자 맞히기 게임 같은 프로그램을 만들 수 있다. 조건이 계속 참이면 무한 루프가 발생할 수 있으므로 종료 조건을 잘 설정해야 한다.
또한 반복문 안에 반복문을 넣는 중첩 반복문도 배운다. 구구단, 별 출력, 표 형태 출력 등에 활용했다.
