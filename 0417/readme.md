## 6 파이썬 자료구조 |: 리스트 요약

리스트는 여러 개의 값을 순서대로 저장하는 자료구조이다. 대괄호 []를 사용해 만든다.
```
scores = [10, 20, 60, 70, 80]
```
리스트의 각 값은 요소라고 하며, 위치 번호인 인덱스를 이용해 접근한다. 인덱스는 0부터 시작한다.
```
scores[0]  # 첫 번째 요소
```
리스트는 값을 변경할 수 있다.
```
scores[0] = 100
```
음수 인덱스를 사용하면 뒤에서부터 접근할 수 있다.
```
scores[-1]  # 마지막 요소
```
리스트에는 다양한 연산이 있다.
```
append()   # 추가
insert()   # 원하는 위치에 삽입
remove()   # 특정 값 삭제
pop()      # 마지막 또는 특정 위치 삭제
sort()     # 정렬
len()      # 길이
max()      # 최댓값
min()      # 최솟값
sum()      # 합계
```
리스트의 일부를 잘라내는 것을 슬라이싱이라고 한다.
```
scores[1:4]
```
또한 리스트 안에 리스트를 넣으면 2차원 리스트를 만들 수 있다. 표, 행렬, 좌석 배치 같은 자료를 표현할 때 사용했다.
이 장에서는 성적을 입력받아 평균, 최댓값, 최솟값, 특정 점수 이상인 학생 수를 구하는 프로그램을 만들었다.
