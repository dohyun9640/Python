## 2장 변수와 수식 요약

변수는 값을 저장하기 위해 이름을 붙인 메모리 공간이다. 파이썬에서는 변수에 값을 대입하면 자동으로 변수가 만들어진다.
```
x = 100
y = 200
sum = x + y
print(sum)
```
파이썬의 변수는 값을 직접 담는 상자라기보다 메모리에 저장된 객체를 가리키는 이름에 가깝다. 변수값은 다시 대입하면 바뀔 수 있다.

산술 연산자로는 +, -, *, /, //, %, ** 등이 있다.
예를 들어 **는 거듭제곱, %는 나머지, //는 몫을 구한다.

또한 연산자에는 우선순위가 있다. 곱셈과 나눗셈이 덧셈과 뺄셈보다 먼저 계산되며, 괄호를 사용하면 우선순위를 바꿀 수 있다.

사용자 입력은 input() 함수로 받는다. 입력값은 기본적으로 문자열이므로 숫자 계산을 하려면 int()나 float()로 형 변환해야 한다.
```
x = int(input("숫자를 입력하세요: "))
```
이 장에서는 BMI 계산, 거듭제곱 계산, 거스름돈 계산 같은 예제를 통해 변수, 입력, 출력, 연산식을 연습했다.
