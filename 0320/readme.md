## 3장 조건문 요약

조건문은 프로그램이 상황에 따라 다르게 동작하도록 만드는 제어문이다. 조건문이 없다면 프로그램은 항상 같은 순서로만 실행된다.

기본 구조는 if, else, elif이다.
```
if score >= 60:
    print("합격")
else:
    print("불합격")
```
조건식에는 관계 연산자가 사용된다.
대표적으로 >, <, >=, <=, ==, !=가 있다.

여러 조건을 함께 판단할 때는 논리 연산자를 사용한다.
```
and   # 둘 다 참이면 참
or    # 하나라도 참이면 참
not   # 참과 거짓을 반대로 바꿈
```
파이썬에서는 들여쓰기로 코드 블록을 구분한다. 따라서 조건문 안에서 실행될 문장은 반드시 같은 깊이로 들여쓰기해야 한다.

조건문은 단순 if문, if-else문, 중첩 if문, 연속 if-elif-else문으로 확장할 수 있다. 예제에서는 할인 가격 계산, 지진 규모에 따른 피해 설명, 숫자 비교 등을 다뤘다.
