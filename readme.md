## 1장  파이썬 소개 요약

컴퓨터는 하드웨어와 소프트웨어로 구성되며, 다양한 일을 할 수 있는 유연한 기계이다. 컴퓨터에게 일을 시키려면 사람이 자세한 명령어 목록을 작성해야 하는데 이것을 프로그램이라고 한다. 인공지능, 주식 자동매매, 로봇 프로세스 자동화처럼 여러 분야에서 프로그램이 활용된다. 컴퓨터는 반복적이고 지루한 작업을 빠르고 정확하게 처리하는 데 강점이 있다. 컴퓨터는 사람의 언어를 직접 이해하지 못하므로 프로그래밍 언어가 필요하다. 사람이 파이썬 같은 언어로 프로그램을 작성하면 컴파일러나 인터프리터가 이를 컴퓨터가 이해할 수 있는 기계어로 바꿔준다. 문법이 간결하고 배우기 쉬우며 결과를 바로 확인할 수 있는 인터프리터 언어라 초보자에게 적합하다. 또한 라이브러리가 풍부하고 설치가 쉬워 데이터 분석, 인공지능, 웹 개발 등 다양한 분야에 활용된다. 주피터 노트북과 Colab은 코드와 설명을 함께 작성하며 실행할 수 있는 환경이다. 기초 문법으로는 print() 함수, 문자열, 계산식 출력이 다뤄진다. 문자열은 큰따옴표나 작은따옴표 안에 작성해야 하며 따옴표가 없으면 오류가 발생한다. print() 함수는 문자열이나 계산 결과를 화면에 출력할 때 사용한다.

예를 들어:

`print("파이썬에 오신 것을 환영합니다.")`

`print("2+3=", 2+3)`

오류 처리에서는 따옴표 누락, 대소문자 오류, 괄호 오류 등이 소개된다. 파이썬은 대소문자를 구분하므로 print와 Print는 다르게 인식된다. 소스 파일의 확장자는 .py이다.

## 2장 변수와 수식

변수는 값을 저장하기 위해 이름을 붙인 메모리 공간이다. 파이썬에서는 변수에 값을 대입하면 자동으로 변수가 만들어진다.

x = 100

y = 200

sum = x + y

print(sum)

파이썬의 변수는 값을 직접 담는 상자라기보다 메모리에 저장된 객체를 가리키는 이름에 가깝다. 변수값은 다시 대입하면 바뀔 수 있다.

산술 연산자로는 +, -, *, /, //, %, ** 등이 있다.
예를 들어 **는 거듭제곱, %는 나머지, //는 몫을 구한다.

또한 연산자에는 우선순위가 있다. 곱셈과 나눗셈이 덧셈과 뺄셈보다 먼저 계산되며, 괄호를 사용하면 우선순위를 바꿀 수 있다.

사용자 입력은 input() 함수로 받는다. 입력값은 기본적으로 문자열이므로 숫자 계산을 하려면 int()나 float()로 형 변환해야 한다.

x = int(input("숫자를 입력하세요: "))

이 장에서는 BMI 계산, 거듭제곱 계산, 거스름돈 계산 같은 예제를 통해 변수, 입력, 출력, 연산식을 연습했다.

## 3장 조건문

조건문은 프로그램이 상황에 따라 다르게 동작하도록 만드는 제어문이다. 조건문이 없다면 프로그램은 항상 같은 순서로만 실행된다.

기본 구조는 if, else, elif이다.

if score >= 60:
    
    print("합격")

else:
    
    print("불합격")

조건식에는 관계 연산자가 사용된다.
대표적으로 >, <, >=, <=, ==, !=가 있다.

여러 조건을 함께 판단할 때는 논리 연산자를 사용한다.

and   # 둘 다 참이면 참

or    # 하나라도 참이면 참

not   # 참과 거짓을 반대로 바꿈

파이썬에서는 들여쓰기로 코드 블록을 구분한다. 따라서 조건문 안에서 실행될 문장은 반드시 같은 깊이로 들여쓰기해야 한다.

조건문은 단순 if문, if-else문, 중첩 if문, 연속 if-elif-else문으로 확장할 수 있다. 예제에서는 할인 가격 계산, 지진 규모에 따른 피해 설명, 숫자 비교 등을 다뤘다.

## 4장 반복문

반복문은 같은 작업을 여러 번 실행할 때 사용한다. 반복적이고 단순한 작업은 컴퓨터가 매우 잘 처리하므로 자동화에 적합하다.

반복문에는 크게 두 가지가 있다.

for문은 반복 횟수를 알고 있을 때 사용한다.

for i in range(5):

    print("방문을 환영합니다!")

range() 함수는 일정한 범위의 숫자를 만들어준다.

range(5)        # 0부터 4까지

range(1, 6)     # 1부터 5까지

range(1, 10, 2) # 1, 3, 5, 7, 9

while문은 조건이 참인 동안 반복한다.

while count < 5:

    print(count)
    
    count += 1

반복문에서는 누적합, 카운트, 숫자 맞히기 게임 같은 프로그램을 만들 수 있다. 조건이 계속 참이면 무한 루프가 발생할 수 있으므로 종료 조건을 잘 설정해야 한다.
또한 반복문 안에 반복문을 넣는 중첩 반복문도 배운다. 구구단, 별 출력, 표 형태 출력 등에 활용했다.
