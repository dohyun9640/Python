1장 파이썬 소개 요약

컴퓨터는 하드웨어와 소프트웨어로 구성되며, 다양한 일을 할 수 있는 유연한 기계이다. 컴퓨터에게 일을 시키려면 사람이 자세한 명령어 목록을 작성해야 하는데, 이것을 프로그램이라고 한다. 프로그래밍은 이제 이공계뿐 아니라 인문사회계에서도 필요한 기본 기술이 되었다. 인공지능, 주식 자동매매, 로봇 프로세스 자동화처럼 여러 분야에서 프로그램이 활용되기 때문이다. 컴퓨터는 반복적이고 지루한 작업을 빠르고 정확하게 처리하는 데 강점이 있다. 컴퓨터는 사람의 언어를 직접 이해하지 못하므로 프로그래밍 언어가 필요하다. 사람이 파이썬 같은 언어로 프로그램을 작성하면 컴파일러나 인터프리터가 이를 컴퓨터가 이해할 수 있는 기계어로 바꿔준다. 파이썬은 1991년 귀도 반 로섬이 만든 대화형 프로그래밍 언어이다. 문법이 간결하고 배우기 쉬우며, 결과를 바로 확인할 수 있는 인터프리터 언어라 초보자에게 적합하다. 또한 라이브러리가 풍부하고 설치가 쉬워 데이터 분석, 인공지능, 웹 개발 등 다양한 분야에 활용된다. 파이썬 개발 환경으로는 아나콘다, 스파이더, 주피터 노트북, 구글 Colab 등이 소개된다. 아나콘다는 여러 라이브러리가 포함된 배포판이고, 스파이더는 파이썬 개발 도구이다. 주피터 노트북과 Colab은 코드와 설명을 함께 작성하며 실행할 수 있는 환경이다. 파이썬 실행 방식에는 두 가지가 있다. 대화형 모드는 콘솔에서 한 줄씩 입력하고 바로 실행하는 방식이고, 스크립트 모드는 파일에 코드를 저장한 뒤 한 번에 실행하는 방식이다. 코드가 길거나 복잡해지면 스크립트 모드가 더 적합하다. 기초 문법으로는 print() 함수, 문자열, 계산식 출력이 다뤄진다. 문자열은 큰따옴표나 작은따옴표 안에 작성해야 하며, 따옴표가 없으면 오류가 발생한다. print() 함수는 문자열이나 계산 결과를 화면에 출력할 때 사용한다.

예를 들어:
print("파이썬에 오신 것을 환영합니다.")
print("2+3=", 2+3)

오류 처리에서는 따옴표 누락, 대소문자 오류, 괄호 오류 등이 소개된다. 파이썬은 대소문자를 구분하므로 print와 Print는 다르게 인식된다. 소스 파일의 확장자는 .py이다.
